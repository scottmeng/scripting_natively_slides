how does it work?

  1. Injection of Android object in V8 runtime
  2. Use Reflection in compilation time and store the mapping in MetaData
  3. Javascript VM API to invoke C++ code
  4. JNI to invoke Java object instantiation
  5. creating Javascript objects proxying actual Java object
  6. data being marshalled to Javascript object


Performance compromise: 10%
